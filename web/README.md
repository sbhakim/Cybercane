# CyberCane Web

Next.js frontend for CyberCane. Primary docs live in `README.md` and
`docs/OVERVIEW.md` at the repo root.

## Run
```bash
npm install
npm run dev
```

## Environment
- `NEXT_PUBLIC_API_URL` (browser)
- `INTERNAL_API_URL` (server-to-server)

Node 20.18.x is recommended for local dev.
